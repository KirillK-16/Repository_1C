Предназначен для хранения информации о замерах времени по ключевым операциям.

В списке выводятся следующие реквизиты:

  • Дата замера UTC - время окончания замера в часовом поясе UTC;
  • Дата записи - время записи замера в часовом поясе пользователя;
  • Значение - длительность выполнения ключевой операции;
  • Имя - ключевая операция замера;
  • Информационная база - информационная база замера ключевой операции;
  • Минимально допустимый уровень - минимально допустимый уровень APDEX ключевой операции;
  • Номер сеанса - номер сеанса пользователя;
  • Пользователь - пользователь, выполнявший ключевую операцию;
  • Приоритет - приоритет ключевой операции;
  • Целевое время - целевое время выполнения ключевой операции;
