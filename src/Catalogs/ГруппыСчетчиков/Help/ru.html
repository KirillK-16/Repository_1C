Предназначен для хранятся сведения о различных собираемых счетчиках.

В списке выводятся следующие реквизиты:

  • Счетчик - наименование счетчика;
  • Диаграмма - гиперссылка быстрого перехода для просмотра собранных значений счетчика;
  • Контроль "Вкл" - флаг выполнения контроля поступления данных счетчика;
  • Контроль "Допустимо нет данных, мин" - период времени в минутах при превышении которого будет создаваться инцидент с типом "НетДанных";
  • Текущее значение "Среднее" - среднее значение счетчика;
  • Текущее значение "Число срабатываний" - количество значений счетчика;
  • Текущее значение "Сумма" - сумма значений счетчика;
  • Текущее значение "Дата записи" - время записи значения счетчика;
