Задание планировщика описывает, на каких машинах и по какому расписанию будет выполняться заданное действие.

В задании планировщика указывается действие для выполнения, контур администрирования и список агентов (машин), на которых предполагается выполнение данного задания. Также задается расписание задания.

 

Расписание задается в форме редактирования расписания регламентного задания и может быть задано с большой точностью.

 

Для сравнения: расписание плановых задач через Календарь задач администрирования задается с точностью до дня.

 

После записи задания для агентов, для которых флаг «Запущено» в форме задания установлен, начнется выполнение соответствующего действия по расписанию. Фактически по расписанию будут формироваться и выполняться экземпляры соответствующего задания с режимом выполнения «Фоновое». Экземпляры сценариев, созданные в рамках исполнения задания планировщика отмечены в форме списка экземпляров сценариев специальным значком .

 

Журнал выполнения задания можно увидеть в форме элемента задания.

 

 

В форме списка заданий также выводится сводная информация о том, сколько машин выбрано для выполнения задания, для какого числа из них установлен флаг «Запущено», а также количество машин, отмеченных с флагом «Запущено», для которых последний запуск завершился ошибкой.
