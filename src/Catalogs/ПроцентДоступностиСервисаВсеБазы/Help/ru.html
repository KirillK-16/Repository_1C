Процент доступности информационных баз.

Показатель вычисляет процент времени, в течении которого информационные базы, зарегистрированные в ЦКК, доступны для работы пользователей.

Для работы показателя требуется, чтобы была запущена и настроена контрольная процедура "Контроль подключений".

При настройке показателя необходимо выбрать период, который будет учитываться для расчёта значения показателя:

  • минута,
  • час,
  • день,
  • неделя,
  • месяц.

Например, если выбран час, то значение показателя в любой точке будет равно проценту времени за последний час, в течении которого информационные базы, зарегистрированные в ЦКК, доступны для работы пользователей.

При настройке показателя нужно указать список информационных баз, учитываемых при расчёте общего процента доступности.

Весовой коэффициент информационной базы при расчёте процента доступности определяется параметром информационной базы «Число пользователей», значение которого должно быть выбрано, равным максимальному числу пользователей, которые могут пострадать в случае, если информационная база окажется недоступной.

Также можно настроить параметры визуализации замеров на диаграмме на форме "Мониторинг":

  • Настройки вида линии серии на диаграммах (толщина, цвет, тип).
  • Настройки масштаба, используемого при отображении линии серии на диаграммах.
  • Параметры сглаживания замеров (поддерживается сглаживание методами скользящего среднего, линейного аппроксимирования или средствами однослойного персептрона).
