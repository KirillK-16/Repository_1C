"Методика" - это рекомендация по настройке, решению проблем или эксплуатации контролируемого объекта.

При создании нового элемента, в форме отображается подробная инструкция по созданию собственной методики.

Методику можно отправить для согласования по кнопке "Отправить для согласования". После одобрения методики она будет доступна всем пользователям ЦКК.
