Добавление новой методики

Для создание новой методики необходимо создать файл в формате HTML, содержащий текст методики.

С помощью кнопки "Редактировать" (1), выполняется переход в режим редактирования методики.

По кнопке "Обновить с диска" (3) файл HTML прикрепляется к методике. Для сохранения текста методики в файл, необходимо нажать кнопку "Записать на диск" (4).

Если методика содержит изображения или файлы, то необходимо добавить их в таблицу "Изображения и файлы" с помощью кнопки "Добавить" (5). Кнопки (6) и (7) позволяют обновить файл или записать его на диск.

Для помещения ссылок на файлы в текст HTML используется "Идентификатор файла" (8), который заменяет путь к файлу. Например, для добавления изображения в текст, в тэге <img src="путь к файлу"> значение "путь к файлу" должно быть заменено на идентификатор в квадратных скобках: <img src="[87f4fb37-61f7-4b07-ac77-22f3fece8d0f]">

В разделе "Применение" (9), указывается, для чего применима данная методика.

Методику можно отправить в "фирму 1С" для согласования с помощью кнопки "Отправить для согласования" (2). Если методика будет одобрена, то она станет доступна всем пользователям ЦКК.

В поле "Одобрено" (10) указывается дата согласования (или изменения) методики.
