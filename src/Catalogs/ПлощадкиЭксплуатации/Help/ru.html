В площадках эксплуатации хранятся сведения о контролируемых объектах.

В списке выводятся следующие реквизиты:

  • Наименование - полное наименование площадки эксплуатации, группы однотипных объектов контроля, объекта контроля;
  • Тип элемента - тип элемента площадки;
  • Проверено, % - процент выполнения рекомендуемых методик для объекта контроля;

Поддерживается добавление, редактирование, установки пометки на удаление объектов контроля. При установке пометки удаления объект контроля помещается в корзину с возможностью обратного восстановления. При переносе в корзину сбор данных по данному объекту прекращается, редактирование реквизитов недоступно.

Площадки эксплуатации, в зависимости от типа элемента, позволяют настраивать следующий функционал:

  • Оборудование - сбор счетчиков производительности, доступность, использование агента КИП;
  • Кластер 1С - настройка подключения к кластеру 1С для сбора параметров работы;
  • Информационная база - настройка подключений к информационной базе для сбора параметров;
  • Рабочий сервер кластера 1С - настройка параметров для анализа работы рабочего сервера;
  • Контроль потребления памяти - настройка параметров контроля потребения памяти рабочими процессами кластера 1С;
  • Контроль подключений - настройка параметров контроля доступности информационной базы;
  • Контроль производительности - настройка параметров загрузки замеров производительности информационной базы;
  • Контроль регламентных заданий - настройка параметров контроля разрешения/запрета выполнения регламентных заданий;
  • Контроль нагрузочных тестов - настройка параметров контроля выполнения нагрузочных тестов;
  • Оценка пользователей - настройка параметров выполнения опроса качества работы пользователями;
  • Контроль устойчивости системы - настройка параметров сбора дампов платформы;
  • Анализ вызовов кластера - настройка параметров сбора и анализа вызовов кластера по технологическому журналу;
  • Мониторинг системных ошибок - настройка параметров сбора и анализа системных ошибок по технологическому журналу;
  • Публикация - настройка параметров контроля доступности публикации;

 

При нажатии на кнопку "Показать методики", отображается поле дерева методик и поле описания текущей методики.

"Методика" - это рекомендация по настройке, решению проблем или эксплуатации контролируемого объекта. При выборе объекта контроля, снизу отображается дерево методик, которые желательно проверить или применить для данного объекта.

После установки курсора на методику, справа появляется текст с ее описанием. Методику можно пометить как выполненную, нажав соответствующую кнопку.

Список предопределенных методик можно загрузить из Интернет или с диска.
Для добавления своих методик необходимо нажать кнопку "Редактировать".
