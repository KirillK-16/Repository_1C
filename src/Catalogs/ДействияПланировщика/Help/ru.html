Действие планировщика – это тот сценарий, который должен выполняться с определенной периодичностью.

 

 

Действие планировщика характеризуется выполняемым сценарием и его параметрами. Параметры для действия планировщика задаются в явном виде и будут одинаковыми для всех запускаемых на основании этого действия заданий.
