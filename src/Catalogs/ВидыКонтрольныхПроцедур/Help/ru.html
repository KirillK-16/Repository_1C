Виды контрольных процедур

Виды контрольных процедур определяют какие контрольные процедуры могут быть созданы с системе.

Параметры контрольных процедур используются как параметры по умолчанию для контрольных процедур соответствующего вида.

На форме вида контрольной процедуры имеется вкладка Рекомендации, в которой дано детальное описание принципа действия процедуры и значений её параметров.
