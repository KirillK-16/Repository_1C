Введение сущности, отвечающей за разделение на контуры в рамках разных зон ответственности получило развитие в рамках функционала разделения прав доступа к различным контурам администрирования.

В конфигурации «Центр администрирования» есть две роли для работы пользователей АвтоматизацияПолныеПрава, АвтоматизацияОграниченныеПрава. Пользователи с ролью АвтоматизацияПолныеПрава имеют неограниченный доступ ко всем контурам администрирования. Для пользователей с ролью АвтоматизацияОграниченныеПрава могут быть установлены ограничения на работу с контурами администрирования.

Для работы с доступом используется справочник «Ограничения доступа к контурам администрирования».

 

В справочнике есть возможность задать профили доступа, в которых можно указать, для каких пользователей, к каким контурам разрешен доступ.

 

Если пользователь с ролью АвтоматизацияОграниченныеПрава не указан не в одном из профилей доступа, это значит, что ограничения доступа для него не заданы и у него есть доступ ко всем контурам.

Увидеть, для каких пользователей заданы профили доступа, можно в форме списка справочника «Ограничения доступа к контурам администрирования».

Значок закрытого замка сигнализирует о том, что пользователю установлена роль АвтоматизацияОграниченныеПрава (открытый замок - АвтоматизацияПолныеПрава). Значок с шестеренками указывает на то, что пользователь выбран в одном или более профилей доступа. При активизации строки с пользователем ИБ в таблице «Доступные контуры» можно увидеть, какие контуры доступны данному пользователю.
