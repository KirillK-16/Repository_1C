Предназначен для просмотра текущего состояния очереди заданий.

В списке выводятся следующие реквизиты:

  • Наименование - наименование задания;
  • Расписание - расписание выполнения задания;
  • Состояние задания - текущее состояние задания;
  • Запланированный момент запуска, UTC - универсальноые время следующего выполнения задания;
  • Начало крайнего запуска, UTC - универсальное время крайнего запуска задания;
  • Окончание крайнего запуска, UTC - универсальное время завершения крайнего запуска задания;
  • № попытки - номер попытки запуска;
  • Имя метода - метод, выполняющий задание;
