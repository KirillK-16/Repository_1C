ЦКК содержит функциональность по централизованному отслеживанию инцидентов на основе собственных данных и данных прикладных систем, которые контролируемого контура.

 

Пояснения:

Внешние инциденты: ЦКК содержит специальный HTTP-сервис InputIncidentTickets, с помощью которого прикладные решения могут оповещать о возникающих инцидентах прикладного характера. Например, можно централизованно отслеживать с помощью ЦКК работоспособность обмена между прикладными подсистемами или работоспособность журналов регистрации и проч.
