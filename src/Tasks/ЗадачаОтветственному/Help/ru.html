Работа с задачами

После того, как контрольная процедура запущена, ЦКК автоматически выполняет необходимые действия по контролю качества работы системы. В случае если во время контроля обнаружены проблемы, ЦКК автоматически формирует задачи по устранению этих проблем и направляет их ответственным. Если выполнение контрольной процедуры завершается с ошибкой (то есть, контроль невозможен), ЦКК формирует задачи по устранению проблем с контролем и направляет их ответственным за настройку контрольной процедуры.

При создании любой задачи ЦКК оповещает ответственного о необходимости выполнить задачу, отправив ему сообщение по электронной почте и/или при помощи SMS. Кроме того, задача появляется в списке действий, которые необходимо выполнить. Ответственный должен запустить задачу на выполнение, дважды кликнув по ней в списке. Будет открыта форма выполнения текущей задачи. Форма предоставляет детальную инструкцию по выполнению задачи и интерфейс, необходимый для:

  • фиксации факта выполнения действия;
  • фиксации результатов выполнения действия.

Информация о результатах выполнения данного действия может быть использована на следующих шагах выполнения контрольной процедуры.

При получении задачи пользователь для её выполнения должен следовать инструкциям, находящимся во вкладке «Рекомендации» на форме задачи. После выполнения всех действий, указанных в инструкции задачу следует пометить как выполненную.
