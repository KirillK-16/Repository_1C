Предназначен для хранения информации о зарегистрированных в ЦКК дампов платформы.

В списке выводятся следующие реквизиты:

  • Идет обработка - флаг текущего состояния обработки дампа;
  • Дата регистрации - время обнаружения файла дампа;
  • Дата завершения - время полной обработки файла дампа;
  • Объект контроля - оборудование, где обнаружен файл дампа;
  • Вариант дампа - варианта дампа;
  • Размер файла - размер файла дампа;
  • Состояние обработки дампа - текущее состояние обработки дампа;
  • Начало архивирования - время начала архивирования файла дампа;
  • Завершение архивирования - время завершения архивирования файла дампа;
  • Начало экспорта - время начала копирования архива файла дампа в каталог экспорта;
  • Завершение экспорта - время окончания копирования архива файла дампа в каталог экспорта;
  • Описание ошибки - описание ошибки при сбое обработки дампа;
  • Каталог экспорта - каталог в который скопирован архив файла дампа;
  • Номер дампа - порядковый номер варианта дампа;
