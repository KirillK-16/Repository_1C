Отчет "Информация о перезапусках рабочих процессов".

Для работы отчета необходимо, чтобы для кластеров была настроена и запущена контрольная процедура "Контроль потребления памяти".

Отчет предназначен для представления данных количестве активных сеансов на произвольный момент времени. Полученные данные можно использовать для анализа нагрузки на информационную базу.

 

Информация о полях отчета.

  • Поле "Кластер" показывает кластер серверов 1С:Предприятия;
  • Поле "Информационная база" показывает информационную базу;
  • Поле "Количество" показывает количество активных сеансов на период указанный в настройках.

Параметры отчета.

  • Параметр "Период" задает период на который необходимо получить количество сеансов;
  • Параметр "ИБ" позволяет отобрать в отчете только необходимые для анализа информационны базы.
