Отчет предназначен для вывода замеров производительности информационных баз.

  • Настройки

    • Дата начала - начало периода выборки замеров ключевых операций;
    • Дата окончания - окончание периода выборки замеров ключевых операций;
    • Информационная база - информационные базы для выборки замеров ключевых операций;
    • Ключевая операция - ключевые операции для выборки замеров;
    • Пользователь - пользователи для выборки замеров ключевых операций;
