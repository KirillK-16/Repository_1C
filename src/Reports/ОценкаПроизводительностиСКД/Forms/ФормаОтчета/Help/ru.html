Отчет предназначен для анализа производительности информационных баз по методике APDEX, выявления проблем производительности.

  • Настройки

    • Период - основной период выборки замеров ключевых операций;
    • Переодичность - группировка замеров выбранного периода, доступные значения: час, день, неделя, весь период;
    • Период для сравнения - период выборки замеров ключевых операций для сравнения с основным периодом;
    • Информационная база - информационные базы для выборки замеров ключевых операций;
    • Имя ключевой операции - ключевые операции для выборки замеров;
    • Пользователь - пользователи для выборки замеров ключевых операций;
