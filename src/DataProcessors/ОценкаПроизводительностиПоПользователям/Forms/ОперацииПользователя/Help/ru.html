Отчет «Операции пользователя» позволяет оценить производительность системы у выбранного пользователя в разрезе ключевых операций. Для операции отображается значение целевого времени, среднего времени за 7, 30 и 90 дней, а также значение Apdex за те же периоды. Есть возможность скрыть те операции, которые не выполнялись пользователем в указанный период, установить фильтр по нужной операции.

При двойном щелчке по строке таблицы открывается отчет «Ключевая операция по пользователю», детализирующий информацию.
