Отчет «Оценка производительности по пользователям» помогает отвечать на вопросы:

1. у каких пользователей 1С работает быстрее, у каких медленнее?
2. насколько медленно у пользователя работает 1С?
3. какие операции у пользователя работают медленно?
4. как менялась производительность у пользователя во времени?

В таблице на форме отображются пользователи выбранной информационной базы, с указанием общего количества совершенных ими операций за указанный период времени, значения Apdex за указанный и аналогичный предыдущий периоды. Есть возможность установить дополнительные отборы по минимальному количеству операций, совершенных пользователем за период и по динамике (улучшение или ухудшение) Apdex.

При двойном щелчке по строке таблицы открывается отчет «Операции пользователя», детализирующий информацию.
