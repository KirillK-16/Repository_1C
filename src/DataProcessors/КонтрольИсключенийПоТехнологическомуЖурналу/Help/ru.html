Контроль исключений по технологическому журналу

Обработка позволяет посмотреть рейтинг (с группировкой по тексту ошибки) по всем событиям "EXCP", обнаруженным при разборе технологического журнала, упорядоченный по числу ошибок, за выбранный период.

В настройках можно при желании указать период, за который будет выполняться анализ, кластер, рабочий сервер и информационную базу.

Если вы хотите отфильтровывать сообщения об ошибках по тексту ошибки, выберите строчку, не интересующей вас ошибки, в построенном рейтинге и нажмите на кнопку "Игнорировать". Список игнорируемых ошибок находится во вкладке "Игнорируемые исключения".
